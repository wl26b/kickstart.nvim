# CLAUDE.md — neovim config

Personal fork of kickstart.nvim (single-file `init.lua`).

## Working rules
- `origin` = your fork (github.com/wl26b/kickstart.nvim) — commit to `master`,
  push `origin`, no branch/PR. `upstream` = nvim-lua/kickstart.nvim, for syncing
  TJ's updates only.
- After editing config, **commit AND push to origin** — reproducible across
  machines, so an uncommitted edit isn't "done."
